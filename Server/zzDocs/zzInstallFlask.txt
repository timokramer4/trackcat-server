1. Install Python -->32bit
2. Install Xampp  
(3. Install Visual Studio with c++ addon)
4. run "pip install mod_wsgi in terminal" if error, create environment vairable "MOD_WSGI_APACHE_ROOTDIR" --> Path to Apache installation ex: C:/xampp/apache/ --> use forward slashes //
5. run "mod_wsgi-express module-config"
6. copy whole output from 5. in httpd.conf of apache to load the Module
7. add Includes ExecCGI to Directory options
    add .py to AddHandler cgi-script .py in <IfModule mime_module>
8. create Virtualhost

<VirtualHost *:80>
	ServerName dsui
	WSGIScriptAlias / "C:/xampp/htdocs/web.wsgi"  	
	DocumentRoot "C:/xampp/htdocs"
	<Directory "C:/xampp/htdocs/">
			AllowOverride All
	</Directory>
</VirtualHost>


9. create web.wsgi in htdocs/

import sys
sys.path.append('C:/xampp/htdocs')
from main import app as application
